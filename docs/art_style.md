# 美术画风规范 — 皇家骑士团外传复刻

## 一、地图类型与空间透视规则

### 1. 基础透视：45°斜二测伪3D立体方格棋盘

全程采用等轴测45°斜视角（斜二测），纯2D像素堆叠模拟立体高低差，无真实3D建模。每一格是正方形像素地板，视觉呈现菱形棋盘，支持多层海拔落差（高地、悬崖、洼地、高台）。

- 格子逻辑：单格统一尺寸，人物、建筑、地形全部吸附网格；地图存在高低层级遮挡，高处地块会遮盖下层地形/单位，悬崖、台阶、水渠形成垂直落差通路。
- 移动标尺：画面右下角自带MOV彩色移动条，右上角标注STEP剩余移动步数，右侧悬浮属性增减面板，UI与地图完全分层不遮挡地形主体。

### 2. 全场景地图分类

1. **洞窟/地下岩洞地图**：灰蓝色岩石层叠岩壁，深灰石砖地面，深蓝色流水河道（带横向波纹像素动画），冷色调低饱和。
2. **城堡/石砌要塞地图**：青绿色石砖高台城墙，竖井火把（红橙跳动），封闭高台+壕沟洼地，复古中世纪堡垒。
3. **野外平原地图**：嫩绿草地，土黄土路，河流/浅滩分割地块，色调明亮柔和。
4. **山地悬崖地图**：棕褐色层叠岩山，大面积断崖落差，暗棕阴影强化山体厚重感。
5. **室内圣堂/地下室地图**：浅灰大理石地砖，石柱拱门，暖黄光源，低对比肃穆色调。

### 3. 地形像素绘制规范

- 所有地块用2层像素明暗区分体积，单地形不超过3种色阶。
- 水域统一横向重复波纹像素动画，草地用短竖线草丛纹理，岩石用块状凹凸像素勾边。
- 高低差台阶用2-3格高度阶梯过渡，边缘加黑色轮廓线。

## 二、人物像素画风

### 1. 体型比例：Q版迷你二头身像素小人

全体角色统一2头身Q版比例，头大身短，躯干方正，四肢短小，单角色精灵尺寸极小，轮廓粗黑1像素描边。

### 2. 职业视觉区分

- **战士/骑士**：金属盔甲银灰/金色像素块，全覆盖头盔，剑/枪长武器，红蓝短披风。
- **白魔法师**：红白尖顶兜帽长袍，宽松布质，短法杖。
- **弓箭手**：绿褐紧身布衣，短帽，背部弓箭标识。
- **法师/祭司**：紫色宽袍，兜帽遮半张脸。
- **主角精英单位**：紫发/金发精致像素发丝，盔甲高光金边。

### 3. 光影与色彩规则

- 人物固定左上单一光源，所有单位光影方向完全统一。
- 中世纪复古低饱和马卡龙像素色，极少渐变。
- 待机小幅上下浮动动画，头顶标注黑色END文字表示行动结束。

## 三、建筑&场景物件像素画风

- 城堡墙体：模块化石砖拼接，火把动态火焰，窄窗黑块。
- 自然物件：块状岩石，短竖线草丛，横向波纹水体动画。
- UI：米黄羊皮纸背景装备框，HP/MP长条，牛皮纸色底部信息栏。

## 四、整体统一美术画风

### 1. 媒介与像素规格

GBA 16位复古点阵像素画，硬边像素无抗锯齿，1像素纯黑轮廓线，无模糊渐变。

### 2. 美术氛围基调

中世纪欧式战棋复古写实像素风，兼具黑暗奇幻厚重感。分场景统一色调，光影极简。

### 3. 分层逻辑

1. 底层：地形方格地板像素层
2. 中层：地形物件（岩石、河流、草丛、台阶）
3. 上层：角色人物精灵、建筑墙体高台
4. 最顶层：全局UI面板

### 4. 统一绘制强制规则

1. 所有物体必须加1像素纯黑色外轮廓
2. 全局统一左上单向光源，最多3色阶
3. 严格45°斜二测网格，所有内容对齐菱形方格
4. 角色固定2头身Q版迷你像素人
5. 场景分洞窟冷调、城堡青调、野外暖绿调三大色系
6. 动态元素仅水域波纹、火把火焰，其余静态
